# Среда выполнения

- OS: Linux 6.18.5, x86_64
- Python: 3.11.14
- Node.js: 22.22.0
- GCC: 13.3.0
- Диск: 252G, ~31G свободно
- Пользователь: root (внутри контейнера/песочницы)
- Дата первой сессии: 2026-03-10

## Доступные инструменты Claude Code
- Read, Write, Edit — работа с файлами
- Bash — выполнение команд
- Glob, Grep — поиск файлов и содержимого
- WebSearch, WebFetch — доступ к вебу (возможно ограничен)

## Структура проекта
Проект aria-sandbox изначально задуман для локальной LLM (Qwen3-Coder),
но текущая сессия работает через Claude Code (модель Opus 4.6).
Запуск через run_agent.sh, каждая итерация — новая сессия.
