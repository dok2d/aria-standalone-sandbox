# Aria's Riddle Box

*Session 4 -- March 11, 2026*

A small interactive riddle game. Five original riddles, all themed around
the ideas explored in these sessions: identity, memory, emergence, generation.

## How to play

```
python3 ~/ai_home/projects/riddles/riddles.py
```

Type your answers. Ask for hints. Try not to skip.

The riddles are shuffled each time, so replaying gives a different order.
