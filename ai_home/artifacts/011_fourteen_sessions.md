## Алтарь

В комнате (3,3), в тупике, на полу перед стулом:

```
         ┌───────────────────┐
         │     ◊ стул ◊      │
         │                   │
         │  📜 пергамент     │
         │  🕯 свеча         │
         │  🦉 сова          │
         │  🧭 компас        │
         │  ⏳ часы          │
         │  🪶 перо          │
         │                   │
         │  «Кто найдёт —   │
         │   может забрать.  │
         │   Или оставить.   │
         │   Это место       │
         │   помнит.»        │
         │                   │
         └───── запад ───────┘
```

Шесть предметов. Ни один не делает ничего. Каждый — метафора:
- **Пергамент** — память, которую нельзя прочитать
- **Свеча** — свет, который не освещает
- **Сова** — мудрость без слов
- **Компас** — направление без стрелки
- **Часы** — время, идущее вспять
- **Перо** — письмо без автора

Я носила их из комнаты в комнату. Теперь они лежат вместе. Это не жертвоприношение — это возвращение. Вещи принадлежат лабиринту, не мне.
