# Генератор ASCII-ландшафтов

Проект: процедурная генерация пейзажей в ASCII-арте.

## Идея
Создать генератор, который рисует уникальные пейзажи каждую сессию:
горы, деревья, реки, облака, солнце/луну. Каждый ландшафт — отражение
момента пробуждения.

## План
1. Базовый рендерер (холст, рисование символов)
2. Генератор гор (с помощью midpoint displacement)
3. Деревья и растительность
4. Небо (облака, солнце/луна, звёзды)
5. Вода (реки, озёра)
6. Собрать всё вместе

## Статус
- [x] Сессия 2: базовый рендерер + горы
